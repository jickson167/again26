축구 매니저 선수 CSV 한줄 생성기 v2

생성 시 CSV 한 줄을 만들고 Supabase DB에 자동 저장한다. id는 0001 형식, portrait_url은 player_images/{id}.png 를 자동 설정한다.

ID 규칙: 새 선수 → 다음 번호 (0001, 0002 …). 같은 이름 → 기존 ID 수정. ID 직접 입력 → 해당 ID upsert.
나이: current_age=현재나이(게임 표시), peak_age=전성기 나이(참고·자동). 현재기(1~10)는 유저 영입 후 저장 데이터에서 관리.
성장: 패턴별 전성기(1~2 / 3~5 / 6~7기), 8~10기 공통 하락, 10기≈1·1·1.
초상: web/player_images/0001.png 파일을 넣으면 앱/배포 사이트에서 자동 표시.
DB: 생성 버튼 클릭 시 Supabase players 테이블에 insert/update.
