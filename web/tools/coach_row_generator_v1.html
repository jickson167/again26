축구 매니저 감독 CSV 생성기 v1

선수 생성기와 같은 방식입니다. 한 번에 만들지 말고 3단계로 ChatGPT에 CSV를 계속 요구하고, 붙여넣기해서 데이터를 완성합니다.

① 다단계 생성 (추천 · 5~10명)

진행 순서: 0 시드 입력 → 1 프로필/고유능력 → 2 포메이션 적합도 → 3 통솔력곡선/코멘트 → 최종 CSV 생성 → DB 저장

줄마다 이름, 국적, 나이, 랭크, 시드종류 또는 앞에 번호를 붙여도 됩니다.
예: 1,대니 뢸,독일,36,1,일반시드 · 대니 뢸,독일,36,1,일반시드

1️⃣ 프로필 + 스타일 + 고유능력

2️⃣ 모든 포메이션 적합도

3️⃣ 통솔력 곡선 + 코멘트

② 최종 CSV / DB 저장

단계별 CSV를 적용한 뒤 [최종 CSV 생성]을 누르세요.
